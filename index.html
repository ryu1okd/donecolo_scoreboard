Donation Colloseum Scoreboardとは？

twitchにて放送中のDaigo the BeasTVの企画「ドネーションコロシアム」で使わている特殊ポイントルールのスコアをメモする非公式WEBアプリです。

ドネーションコロシアムとは？

格ゲーチェッカーにて詳しく載っています。

ルール説明

  • ・○セット先取の試合形式で行う
  • ・2ポイントで1セット獲得
  • ・2連勝すると1ポイント獲得
  • ・3連勝すると2ポイント獲得(その時点で1セット獲得となる)
例
 Player1Player2 
第1試合○☓
第2試合○☓←P1 + 1pt
第3試合○☓←P1 + 1pt = 2pt => 1set
第4試合☓○
第5試合☓○←P2 + 1pt
第6試合○☓
第7試合☓○
第8試合☓○←P2 + 1pt = 2pt => 1set
